Project Details

Lihn Zhee skull from Star trek Picard

This is one of the toughest projects I have ever worked on, It was done in my early days as a ZBrush user, safe to say I was a complete beginner and it was frustrating in early stages of the project where you want to block things out, but out of the blue I did it!

Purposes of this project

This project was made for 3D printing and that is why it was essential for me to make sure that the output formats supported 3D printing and other properties like the mesh being fully solid with no manifold or loose vertices. Manually this would be a tideous task but I have found a useful add-on in Blender which does this task to the artist in no time.
